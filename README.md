# music-s2-claude-code

本地音乐播放器，纯离线播放。

## 功能特性

- 扫描本地音乐文件
- 音乐播放/暂停/上一首/下一首
- 迷你播放器（显示专辑封面）
- 日志记录（保存在 app 外部存储目录，自动限制大小）
